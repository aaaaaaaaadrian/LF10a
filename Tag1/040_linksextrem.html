Anchor-Tags (<a>) - Links in HTML

Mit dem Anchor-Tags können wir Links erstellen. Links führen zu anderen Seiten, Dateien oder zu einer Stelle auf derselben Seite.

1. Grundaufbau eines Links

So sieht ein einfacher Link im Code aus:

⁢a href="https://google.com/"> Ich bin ein Links </a>

So sieht ein Link im Browser aus:

Google

2. Das Attribut href

href steht für Hypertext Reference. Das Attribut href gibt an, wohin der Link. Ohne href ist es kein richtiger Link.

  • Youtube
  • Moodle
  • Ihhhhh... HK
  • href="./010_hallo.html" -> link zu einer Datei im gleichen Ordner
  • href="#ziel" -> Link zzu einer Stelle auf derselben Seite
  • href="https://google.com/" -> Link der zu einer anderen Website führt

3. Link in einem neuen Tab öffnen

Mit target="_blank" öfffnet sich der Link in einem neuen Tab.

<a href="https://www.google.com/" target="_blank" rel="noopener noreferrer"> Ich bin ein Link</a>
Das ist ein Link

Sicherheit bei target="_blank", das die Zielseite, meine Seite nicht manipulieren kann.
