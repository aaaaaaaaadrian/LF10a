Kontaktformulare in HTML

Formulare helfen uns dabei, Daten von Nutzern einzusammeln. Die wichtigsten Bausteine sind:

  • <form> = der Formularrahmen (container)
  • <label> = Beschriftung für die Eingabefelder
  • <input> = Eingabefeld
  • <textarea> = großes Textfeld
  • <select> und <option> = Dropdown Auswahl
  • <button> = Button

1. Das einfache Kontaktformular

Ein Kontaktformular besteht meistens aus Name, E-Mail und Nachricht.

for (bei <label>) + id
Verbindet Label und Feld. Klickst du auf das Label, springt dein Cursor ins Feld.

2. Wichtige Form-Attribute: action und method

action=""
Wohin die Daten geschickt werden (z.B. server-Adresse oder Skript

method="GET"
Daten werden in die URL geschrieben (gut für eine Suche, schlecht für unsere sensiblen Daten).

method="POST"
Daten werden im Hintergrund gesendet (Standard Formular).

HINWEIS: Ohne echten Server passiert beim Absenden oft nichts Sichtbares, aber HTML "baut" das Formular korrekt.

3. Input-Typen und ihre Funktionen

Der type sagt dem Browser, welche Art von Daten erwartet werden. Dadurch bekommt man die passende Tastaturen am Handy/Tablet und manchmal auch eine automatische Überprüfung.

placeholder=""
Ein Hinweistext im Feld (verschwindet beim Tippen).
min/max
Grenze der Zahlen(oder ein Datum)

4. Pflichtfelder und einfache Validierung

Mit requirted kann man Felder verpflichtend setzen. Der Browser verhindert dann das Absenden, wenn etwas fehlt.

required
Muss ausgefüllt werden

minlength
Mindestlänge für unseren Text.

maxlength
Maximallänge für unseren Text.

5. Auswahlfelder: select, radio, checkbox

radio
Nur eine Option ist auswählbar.

select
Iste eine Auswahlliste zum Ausklappen.

checkbox
Ist variabel zwischen an und aus oder mehreren Auswahlmöglichkeiten.

Kontaktwunsch:


disabled
Kann nicht ausgewählt werden.

selected
Option ist standardmäßig ausgewählt.

6. Buttons: submit, reset, button

type="submit"
Sendet das Formular ab.

type="reset"
Setzt alle Felder auf den Startzustand zurück.

type="button"
Macht erst etwas, wenn JavaScript hinzugefügt wurde.
